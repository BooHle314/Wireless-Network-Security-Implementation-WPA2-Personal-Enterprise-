# Wireless-Network-Security-Implementation-WPA2-Personal-Enterprise-
This project focuses on the design and implementation of secure wireless networks using both WPA2 Personal and WPA2 Enterprise configurations in a simulated environment. The goal was to understand and apply modern wireless security protocols to protect network communication and control access.

#Objectives
Configure and secure wireless networks using industry-standard protocols
Compare WPA2 Personal and WPA2 Enterprise authentication mechanisms
Implement centralized authentication using a RADIUS server
Analyze security implications of different wireless configurations


#Implementation
*WPA2 Personal
-Configured secure wireless access using Pre-Shared Key (PSK) authentication
-Ensured encrypted communication between clients and access points
-Tested connectivity and access control across devices

* WPA2 Enterprise
-Implemented RADIUS-based authentication for centralized user management
-Configured authentication server and integrated it with network devices
-Enabled secure login using unique user credentials
-Simulated enterprise-level wireless security architecture
